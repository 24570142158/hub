# 学习前的小提示
> [!TIP]
> <参数>：必填参数
> <参数1 | 参数2>：参数1和参数2中选一个
>
> [参数]：选填参数
> [参数1 | 参数2]：参数1或参数2或不填

# 创建假人
要创建假人，请在游戏内执行命令：`/bot create <假人名> [皮肤名]`

或在控制台执行命令：`/bot create <假人名> <皮肤名> <世界名> <X> <Y> <Z>`

> [!WARNING]
> <假人名> 需要符合 Minecraft 的玩家名规范

> [!NOTE]
> [皮肤名] 假人将会使用皮肤名对应的皮肤。若没有填写此参数，则假人会使用假人名的对应皮肤。

> [!TIP]
> 如果玩家在创建假人后不离开假人位置，则假人会和玩家发生碰撞而导致一定位置偏移

# 移除假人
假人无法复活，即使游戏开启死亡不掉落，也会在死亡后扔出所有物品并移除自己（而不是像 Carpet 那样断开连接）
> [!TIP]
>在一般情况下未开启假人驻留，假人会在服务器关闭前被全部移除，假人身上的装备会直接消失，所以在重启服务器前请及时拿取

`/bot remove <假人名>`
移除指定的假人

`/bot remove <假人名> [时] [分] [秒]`
定时移除指定的假人

# 假人动作
动作是假人的重要部分，它可以让假人完成更多任务。 截止目前，假人一共拥有 15 种动作

每个假人可以同时执行多个动作。每个动作相互独立，不会互相影响。

> [!WARNING]
> 但不允许左右脑冲突，例如长按右键和连点右键是无效的

`/bot action <假人名> <动作名> <start | stop | list> [其它参数]`
来指定假人的动作

`/bot action <假人名> list`
可查看指定假人当前的所有动作

`/bot action <假人名> stop <动作序号 | all>`
可停止假人当前的 一个 或 所有动作

`/bot action <假人名> start <动作> [动作参数]`
可向假人添加一个动作
